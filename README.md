# 🛒 Python Shopping List App

A simple Command Line Interface (CLI) Shopping List Application built using Python.  
This project allows users to add, view, and remove items from a shopping list interactively.

---

## 📌 Features

- ✅ Add items to shopping list
- ✅ View all added items with numbering
- ✅ Remove items using item number
- ✅ Input validation for incorrect choices
- ✅ Runs continuously until user exits

---

## 🛠️ Technologies Used

- Python 3
- Lists
- Loops (while, for)
- Conditional statements (if-elif-else)
- List methods (`append()`, `pop()`)
